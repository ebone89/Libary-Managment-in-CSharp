# Library Management System

A simple console-based library management application written in C#. Built as a college programming assignment to demonstrate object-oriented programming concepts.

## Features

- Add books to the library
- View all books in the library
- Search for a book by title

## Requirements

- .NET 6.0 or later

## How to Run

### Visual Studio

1. Open the `.slnx` solution file in Visual Studio
2. Press F5 to build and run

### Command Line

```bash
dotnet run
```

## Usage

When the program starts, you will see a menu with four options:

1. **Add a book** - Enter the title, author, ISBN, and publication year
2. **View all books** - Displays every book currently in the library
3. **Search for a book by title** - Enter a full or partial title to search
4. **Exit** - Closes the program

## Concepts Demonstrated

- Classes and objects
- Methods and properties
- List collections
- Console input and output
- Input validation
- Basic error handling with try-catch
